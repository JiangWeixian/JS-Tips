        # 跨文本传输 - postMessage

        跨文档传输，两个步骤。

        1. postMessage
        2. 在对应的文档上设置message事件监听

        分为两种情况：(文中举例了传递iframe的方法)

        1. 如果向同一个域下面的文档发送，例如https://www.baidu.com向https://www.baidu.com/ss
        2. 不同作用域，
        var popup = window.open('https://www.github.com')
popup.postMessage("some messgae", "https://www.github.com")

        具体见教程

        # Drag

        定义了拖拽事件，不需要再用其他方法目的。

        还包括drag元素接触到另外一个元素的时候发生的事件。

        但是有了拖拽事件不代表元素可以拖拽，需要设置draggable="true"来让元素可以拖拽。
