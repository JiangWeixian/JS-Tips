## label作用

**使用方式**

```JavaScript
<label for="Name">Number:</label>
<input type="text" name="Name" id="Name"/>
```

作用在于**聚焦`label`的时候自动聚焦表单控件。**

## autocompelte

设定**输入控件的**`autocomplete=on`可以在下一次输入的时候，自动**提示上一次输入内容。**
