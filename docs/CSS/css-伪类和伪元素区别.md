# 伪类和伪类别
> 如何区分两者

## 共同点

都是在`.css`或者`style`标签中使用。

## 不同点

* 伪类 - `:`前面之后一个冒号，可以视为**某种特别的选择器**

  ```css
  /* 类如 */
  tag:link
  tag:active
  tag:checked
  /* etc */
  ```
* 伪元素 - `::`跟在前面的是两个冒号。和伪类一样：
  ```css
  /* 类如 */
  tag::after
  tag::before
  /* etc */
  ```

就这个两段代码看不出什么区别。

但是如果实际使用过程中，就会发现后面那一段代码会在`html`中添加元素。而前一段代码并不会只会在原有的基础上修改。

**所以伪元素区别很重要的一点：**

1. **伪类**可以通过添加`class`模拟实现，或许过程复杂了一些。
2. **伪元素**要通过在页面中添加元素才能够模拟实现。
