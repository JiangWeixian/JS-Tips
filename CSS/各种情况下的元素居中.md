# Just center
> 各种情况下的元素居中

### 常用的

脑海里面第一反应能够想到的：

对于`block`元素

* 绝对定位
    ```css
    position: absoulte
    margin: ...
    ```
* `flex`是最近用到的比较多的
    ```
    display: flex // 对内部的元素而言
    align-items&just-content
    ```
* `margin: 0 auto`    

还有一些经常用到，但是没有想到可以这么用！

* 对于`inline&inline-block`的元素。设置`line-height&height`一样的时候可以实现垂直居中，`text-align`实现水平居中。

很少用到

* `translate`的百分比是根据自身大小的。而不是父类大小。结合`position`使用就可以实现定位。
