        # TS-Interface

        # 任意数量的类型

        interface SquareConfig {
  color?: string;
  width?: number;
  [propName: string]: any;
}

        只要不是color or width，那么就可以插入属性，且不需要指定类型

        如果是索引那么就是string > number

        # class implements

        实现类

        interface yy-interface {
  () : boolean // 只是举例说明
}

        这样目的就是以函数作为类的接口

        如果是构造器函数那么那么就是new ()作为interface一项。

        class xx implements yy-interface {
  
}

        # class extends

        interface可以通过extends继承。

        假设含有定义class xx，可以通过

        interface a extends class {}

        实现接口a得到class xx内部所有属性
