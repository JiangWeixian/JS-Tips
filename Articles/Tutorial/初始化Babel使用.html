    # Babel初始化教程

    # Node - 没有HTML

    初衷是我只想使用JS文件，没有HTML。

    因此也就没有使用Webpack。

    1. 创建.babelrc文件

    2. npm install babel-presets-env babel-presets-stage-2 --save-dev

    3. npm install --save-dev babel-cli由于没有全局安装，所以没有办法babel xx.js进行es6->es5转换

    4. 打开package.json，在scripts项目添加

      "scripts:" {
  "babel": "babel src -d lib"
}

      含义为将src下所有文件转换为lib(es5)

      具体文件夹根据具体项目转换。

    没有webpack你只能手动。目前没找到单文件转换的npm命令格式。
